```
public static void bubbleSort(int[] arr) {
    if (arr ==null) return;
    for (int i = 0;i < arr.length-1;i++) {
        for (int j = 0 ;j < arr.length-1-i; j++) {
            if (arr[j] > arr[j+1]) {
                swap(arr,j,j+1);
                System.out.println(Arrays.toString(arr));

            }
        }
        System.out.println("#############");
        System.out.println(Arrays.toString(arr));
    }
}
```  

进行优化:  
如果进行到某一趟没有发生交换，说明数组已经排好序，此时就可以退出循环
```
public static void bubbleSort2(int[] arr) {
        if (arr ==null) return;
        for (int i = 0;i < arr.length-1;i++) {

            boolean isSwap = false;
            for (int j = 0 ;j < arr.length-1-i; j++) {
                if (arr[j] > arr[j+1]) {
                    swap(arr,j,j+1);
                    isSwap = true;
                    System.out.println(Arrays.toString(arr));
                }
            }
            if (!isSwap) {
                break;
            }
            System.out.println("#############");
            System.out.println(Arrays.toString(arr));
        }
 }

```
